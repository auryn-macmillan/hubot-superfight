# hubot-superfight

Make users fight!

See [`src/superfight.coffee`](src/superfight.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-superfight --save`

Then add **hubot-superfight** to your `external-scripts.json`:

```json
[
  "hubot-superfight"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
